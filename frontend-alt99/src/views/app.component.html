SQL Query Builder

Fluent API for building complex SQL queries programmatically.

Query Builder API

// Fluent SQL builder
sql_builder_t query = sql_builder_select("*")
    .from("users")
    .where("age > ?", 18)
    .order_by("name", ASC)
    .limit(10);

const char* sql = sql_builder_finalize(&query);
// SELECT * FROM users WHERE age > ? ORDER BY name ASC LIMIT 10

Supported Operations

  • SELECT with column selection and wildcards
  • INSERT with value binding
  • UPDATE with WHERE clauses
  • DELETE with conditions
  • JOIN operations
  • Parameter binding for SQL injection prevention

Event Bus

Pub/sub event system for decoupled communication between components.

Usage Example

// Subscribe to events
event_subscribe(events, "app.started", on_app_event, user_data);

// Emit events
event_emit(events, "app.started", "Application initialized");

// Get listener count
int count = event_get_listener_count(events, "app.started");

Features

  • Thread-safe event publishing
  • Multiple subscribers per event
  • Custom event payloads
  • User data passing to handlers
